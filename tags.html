TAGS





There are many tags in html

somes are given below 

🔹 Basic Tags (Main Structure)


HTML document ki shuruaat <!DOCTYPE html> se hoti hai jo browser ko batata hai ki ye HTML file hai. 
Iske baad <html> tag pura webpage ko wrap karta hai. 
<head> section me title aur meta information hoti hai jo browser me directly visible nahi hoti. 
<body> ke andar wo sab content likha jata hai jo user ko webpage par dikhta hai.


🔹 Heading & Paragraph Tags


HTML me <h1> se <h6> tak heading tags hote hain jisme h1 sabse bada aur h6 sabse chhota hota hai. 
Paragraph likhne ke liye <p> tag ka use hota hai. 
Line break dene ke liye <br> aur horizontal line ke liye <hr> tag use kiya jata hai.


🔹 List Tags



List banane ke liye do tarah ke tags hote hain: 
<ul> (unordered list) jisme bullet points hote hain aur 
<ol> (ordered list) jisme numbering hoti hai. 
In dono ke andar <li> tag ka use karke list items likhe jate hain.


🔹 Anchor & Image Tags


Link banane ke liye <a> (anchor tag) ka use hota hai jisme href attribute hota hai. 
Image add karne ke liye <img> tag use hota hai jisme src attribute image ka path batata hai aur alt attribute description deta hai.


🔹 Table Tags


Table banane ke liye <table> tag use hota hai. 
Iske andar <tr> (table row), <th> (table heading) aur <td> (table data) use kiye jate hain. 
<caption> table ka title dikhane ke liye hota hai.


🔹 Form Tags


Form banane ke liye <form> tag use hota hai. 
User input lene ke liye <input> tag use hota hai jisme type attribute (text, password, email, radio, checkbox) hota hai. 
<label> input ka naam batata hai, 
<textarea> bada text likhne ke liye aur 
<select> aur <option> dropdown banane ke liye use hote hain.


🔹 Semantic Tags


Semantic tags webpage ko structured aur meaningful banate hain. 
Jaise <header> top section ke liye, 
<nav> navigation links ke liye, 
<section> alag-alag content parts ke liye, 
<article> independent content ke liye aur 
<footer> bottom section ke liye use hota hai.


🔹 Other Important Tags (Short Summary)


Kuch aur useful tags hain jaise <div> (container), 
<span> (inline container), 
<strong> (bold text), 
<em> (italic text), 
<pre> (formatted text), 
<main>, <aside> etc. 
Ye sab webpage ko better structure aur styling dene me help karte hain.
